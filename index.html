By yafan90

发表于 2015-03-27

题目描述：

给定一个01组成的串S（比如：”011100011”），长度>1<=S.len<=50，对S进行方法为F的编码得到Q，F描述为S中的每>个数字Q[i]有

Q[i]=S[i-1]+S[i]+S[i+1] ，
Q[0]=0 + S[i]+S[i+1]，
Q[len-1] = S[len-2]+S[len-2]+0，

即边界处不存在的数字用0代替，
例如：

F{"011100011" } = "123210122" ;
F{"01101001101101001101001001001101001"} = "12221112222221112221111111112221111"。

输入输出：

先给出一个加密串Q ，分别给出S[0]=’0’ 和S[0]=’1’时对应的原数>字串，如果无法成功解码，输出“NONE”。

样例：

0)”123210122”
Returns: { “011100011”, “NONE” }

1)"11"
Returns: { "01",  "10" }

2)"22111"
Returns: { "NONE",  "11001" }

3)"123210120"
Returns: { "NONE",  "NONE" }

4)"3"
Returns: { "NONE",  "NONE" }

5) "12221112222221112221111111112221111"
Returns: 
{ "01101001101101001101001001001101001",
  "10110010110110010110010010010110010" }

思路：

由Q[i]=S[i-1]+S[i]+S[i+1]
且S[0-1]=0，S[0] =0 或者1
则S [1]=Q[0]-S[0]-S[0-1]
依次：S[i]=Q[i-1] - S[i-1] - S[i-2]
即在确定S[0]的情况下，可以依次计算出S[i]，
这样每次检查算出的S[i]是否合法即可，
注意len=1的情况。
